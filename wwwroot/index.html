Revit Cloud Worksharing Model Migrator


About This Application

Purpose: This application demostrates automatically batch migrating Revit Cloud Worksharing (RCW) models among different hubs/projects, based on the folder.

Problem Solved: Eliminates the manual, Revit Desktop based, time-consuming process of migrating legacy Revit Cloud Worksharing (RCW) models from orignal BIM 360 project to target ACC project. Also supports migration across multiple Revit versions (2025, 2026) with automated version-specific processing through Autodesk Platform Services Revit Automation.

Setup Activity:

  1. Open Configure button to create your AppBundle and Activity
  2. Click delete button to delete the existing AppBundle and Activity
  3. Select the appbundle and Revit version(2025 or 2026), Click Create/Update button to create the AppBundle and Activity

How to Migrate:

  1. Select source folder from the left panel
  2. Select destination folder from the right panel
  3. Click Upgrade to start RCW migration

Important:

  • Ensure that destination project is compatible with the selected Revit version.
  • Create a new empty folder in destination project for the migration, do not use existing folder.

    AppBundle & Activity

    Define AppBundle & Activity only once.
    Redefine only when your plugin code change (creates a new version).
    Tip: Make sure .ZIP bundles are placed at /bundles/ folder
    For this sample the .ZIP name is used as suffix to define AppBundle and Activity names. Activities will have the following parameters:
        rvtFile: Input Revit model
        resultrvt: Output upgraded revit Rvt file
        resultrfa: Output upgraded revit Rfa file
        resultrte: Output upgraded revit Rte file

    Step-by-step guide

    To access your ACC or BIM 360 Docs files with this app, you need to connect your ACC or BIM 360 account with the app.

    1. Log in to ACC or BIM 360 Account Administration . Note that you need account administrator permissions.
    2. If you have more than one account, select the account you want to integrate with the app.
    3. From the toolbar, select SETTINGS > Apps & Integrations.
    4. Click the Add Integration button.
    5. Select BIM 360 Account Administration and BIM 360 Docs, and click Next.
    6. Select I’m the developer, and click Next.
    7. In the APS Client ID and App Name fields, enter the APS client ID and app name provided by the app retailer.
    8. Select the I have saved the Account ID information securely checkbox.
    9. Click Save.

      10. Congratulations! You can now access your ACC or BIM 360 Docs files.
